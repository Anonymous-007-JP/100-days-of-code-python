# 100 Days of Code – Python

This repository logs my day-by-day Python coding journey. It follows Angela Yu’s “100 Days of Code” Udemy bootcamp, building my skills for a data analyst/DSML career.

## Structure

- Day01_Basics: learning how to print and assign data to variables and use input()
- Day02_DataTypes: Data types, Type error, checking conversion, mathematical operation, number manipulation, Made a Tip calculator Project.
- Day03_Functions: if else, Modulo, Nesting and Elif, Multiple ifs, Python pizza order project using if else, Logical Operator, Treasure Island Project.
- Day04_Randomisation_and_Lists: Random module, Python Lists, IndexErrors, nested lists. Project: Rock Paper Scissors Game, Banker Roulette.
- Day05_Python_Loops: For loops with lists, range() function, code blocks and indentation. Project: PyPassword Generator, FizzBuzz challenge.
